![OG Image](public/images/og.png)

## 기술 스택

| 분류         | 기술                     |
| ------------ | ------------------------ |
| 프레임워크   | Next.js 16.2.4           |
| UI           | React 19, Tailwind CSS 4 |
| 언어         | TypeScript 5             |
| 이미지 저장  | html-to-image            |
| 데이터베이스 | Supabase                 |
| 호스팅       | Vercel, Cloudflare       |
| 분석         | Google Analytics         |
| 폰트         | DNFBitBitv2 (픽셀 폰트)  |

## 로컬 실행

```bash
pnpm install
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.
